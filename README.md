# Python Projects 🐍

A collection of beginner Python projects built to learn core programming concepts.

---

## Projects

### 1. Guessing Game 🎯
A number guessing game where the computer picks a random number and the user guesses it with hints.
- **Skills:** Random module, while loop, if/elif/else

### 2. Rock Paper Scissors ✂️
Classic Rock Paper Scissors game played against the computer.
- **Skills:** Random module, functions, if/elif/else

### 3. Calculator 🧮
Performs addition, subtraction, multiplication, and division.
- **Skills:** Functions, if/elif/else, arithmetic operations

### 4. BMI Calculator ⚖️
Calculates Body Mass Index and displays health category.
- **Skills:** Functions, float input, f-strings

### 5. Unit Converter 📏
Converts between km/miles, kg/pounds, Celsius/Fahrenheit.
- **Skills:** Functions, mathematical operations

### 6. Todo List ✅
Add, view, and delete tasks from the command line.
- **Skills:** Lists, functions, while loop

### 7. Contact Book 📒
Add, search, view, and delete contacts.
- **Skills:** Dictionaries, functions, CRUD operations

### 8. Hangman 🎮
Word guessing game with limited attempts.
- **Skills:** Lists, string manipulation, random module

### 9. Tic Tac Toe ❌⭕
Two-player Tic Tac Toe in the terminal.
- **Skills:** 2D lists, functions, win condition logic

### 10. Quiz Game 🧠
Multiple choice quiz with score tracking.
- **Skills:** Dictionaries, lists, score logic

### 11. Flashcard App 🃏
Practice questions and answers with flashcards.
- **Skills:** Dictionaries, functions, while loop

### 12. Quote Generator 💬
Displays a random motivational quote on every run.
- **Skills:** Lists, random module, f-strings

---

## How to run any project
```bash
python filename.py
```
